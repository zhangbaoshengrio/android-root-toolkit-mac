## ✨ 主要功能 (Features)

* **傻瓜式操作**：双击运行，无需配置 Python 环境。
* **镜像提取**：基于 `payload-dumper-go`，从全量包自动提取 `init_boot.img`。
* **自动化流**：自动推送文件到手机 -> 等待 Magisk 修补 -> 自动拉取 -> 自动刷入。
* **安全引导**：内置脚本自动修复 macOS 的权限与 Gatekeeper 拦截问题。
